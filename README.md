# Detection of Malaria in Red Blood Cells using Machine Learning

## Project Overview

Malaria is a life-threatening disease caused by parasites transmitted through infected mosquitoes. Early and accurate diagnosis is essential for effective treatment.

This project uses Machine Learning techniques to classify microscopic images of Red Blood Cells (RBCs) as **Parasitized** or **Uninfected**, helping automate the malaria detection process.

> **Note:** This project was developed as part of our final-year academic project in collaboration with a teammate.


## Features

- Detects malaria-infected red blood cells from microscopic images.
- Binary classification:
  - Parasitized
  - Uninfected
- Image preprocessing and feature extraction.
- Machine Learning model for prediction.
- Performance evaluation using standard classification metrics.
